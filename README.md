## Installation

Use the package manager npm to install de dependencies:

```bash
npm install
npm run dev
```

Or the yarn:

```
yarn 
yarn dev
```

## Project Technologies
ReactJS, Redux, NodeJS, MongoDB, Mongodb Atlas.
